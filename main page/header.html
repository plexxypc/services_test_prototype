What is a poker script?

A poker script is a computer program that contains full poker game source code for front-side written in HTML5, React Js and Unity 3D and the core backend algorithms written in Node Js, Angular Js, MongoDB, and Socket. It is capable of launching online poker games for various platforms, including Browser, iOS, Android, and Facebook. The pokerscript includes three default game varieties: Texas Holdem, Omaha, and Omaha Hi-Lo. This open source pokerscript will be very much helpful for developers who want to make customized card games, poker networks, and re-skinned poker themes.
